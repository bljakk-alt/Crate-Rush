# CrateRush

CrateRush is a World of Warcraft War Mode crate intelligence addon focused on accurate crate lifecycle tracking, prediction, synchronisation, telemetry collection, and battlefield awareness.

The project is built around documented architecture, strict ownership boundaries, lifecycle contracts, protocol specifications, and feature design guidelines. The goal is to create a maintainable platform for crate hunting rather than a collection of disconnected features.

---

## Project Goals

- Accurate crate lifecycle tracking
- Reliable crate prediction
- Group coordination and synchronisation
- Data driven improvement through telemetry
- Clean separation of responsibilities
- Long term maintainability
- Documentation driven development
